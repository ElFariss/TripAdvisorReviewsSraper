# TripAdvisor Reviews Scraper

A simple Python-based web scraper that extracts user reviews from TripAdvisor.

## Features

- Scrapes reviews for a specified location or business on TripAdvisor.
- Extracts review text, rating, author, and review date.
- Outputs results in structured format (e.g. DataFrame or CSV).

### Prerequisites

- Python 3.7+
- Jupyter Notebook
- Required packages:
  - `requests`
  - `BeautifulSoup` (`bs4`)
  - `pandas`
  - `re`

Install the packages using pip:

```bash
pip install requests beautifulsoup4 pandas
